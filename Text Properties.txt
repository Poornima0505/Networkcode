3.Text Decoration 

text-decoration:overline;  (horizontal line above the content)
text-decoration:underline; (horizontal line below the content)
line through;
none;

h1{
    color:deeppink;
    text-align:center;
    text-decoration: overline;
}

.head1{
    color:#8b0850;
    text-align:right;
    text-decoration: underline;

}
p{
    color:rgb(0, 0, 255)
}

span{
    text-decoration: line-through;
}
a{
    text-decoration: none;
}

4. Text Transformation

text-transform:uppercase;

text-transform:lowercase;

text-transform:capitalize;

h1{
    color:deeppink;
    text-align:center;
    text-transform: uppercase;
   
}

.head1{
    color:#8b0850;
    text-align:right;
    text-transform: lowercase;
    
}
p{
    text-transform: lowercase;
    color:rgb(0, 0, 255)
}

Text spacing

1.text-indent

    text-indent: 40px;
    color:rgb(0, 0, 255)
}

2.Letter Spacing- (space will be provided for every letter)

Positive value
Negative Value


3.Line height (Decide the gab between content )

---------------
---------------
---------------

    text-indent: 40px; 
    letter-spacing: 1px;
    line-height:0.9;
    color:rgb(0, 0, 255)
}

4.Word Spacing

p{
    text-indent: 40px; 
    letter-spacing: 1px;
    line-height:0.9;
    word-spacing: 20px;
    color:rgb(0, 0, 255)
    
}
