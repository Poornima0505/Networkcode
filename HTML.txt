Internet - Global Network which connect billions of devices. 

WWW - World Wide Web (Public  Websites Which able to access in our local system through the internet)

Websites - Collection of Webpages

Web Browser - The web browser is an software application to explore world wide web.

Web server -  The web server are designed to store,process and send information over www.

Two Types of Websites

Static Website  - Consistent(same content delivered to the users) 

Dynamic Website - Dynamic Website retrieves content in real-time,allowing it alter with the user.
